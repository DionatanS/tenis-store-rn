# Loja de Tênis

Este projeto é uma simulação de uma loja de tênis mobile utilizada para estudos praticos de react-native com expo, com a intenção de agregar mais conhecimentos desse framework.

## Executar o projeto:
- **`yarn install`**
- **`yarn start`**
